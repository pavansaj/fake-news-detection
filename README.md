# 📰 Fake News Detection
Fake News Detection using ML

## 📌 Project Overview
This project detects whether a news article is fake or real using Machine Learning techniques.

## 🎯 Objective
- Identify fake news from real news  
- Use NLP and ML algorithms for classification  

## 🧠 Technologies Used
- Python  
- Pandas  
- NumPy  
- Scikit-learn  
- Natural Language Processing (NLP)  

## ⚙️ How It Works
1. Data collection  
2. Data preprocessing  
3. Feature extraction (TF-IDF)  
4. Model training  
5. Prediction
